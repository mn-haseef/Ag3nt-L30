## Functional Requirements
Modules used (while installation):
* sklearn 0.24.1
* pyfiglet 0.8.post1
* pefile
* joblib 1.1.0

## Installation
Clone the repo
``` https://github.com/mn-haseef/Ag3nt-L30.git ```

After clone this repo
``` cd Ag3nt-L30 ```

To install dependencies 
``` pip install -r requirement.text ```

To run the tool
``` python3 Ag3nt-L30.py ```
